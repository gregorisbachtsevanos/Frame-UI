## 🚀 Quick Start

Frame UI makes it easy to build consistent, accessible user interfaces. Get started in minutes with our comprehensive component library.

### Installation

Install Frame UI using your preferred package manager:

```bash
npm install @frame-ui/ui @frame-ui/theme
# or
yarn add @frame-ui/ui @frame-ui/theme
# or
pnpm add @frame-ui/ui @frame-ui/theme
```

### Basic Usage

Import components directly into your React application:

```tsx
import { Button, Card, Input } from '@frame-ui/ui';
import { vars } from '@frame-ui/theme';

export function App() {
  return (
    <Card>
      <h1>Welcome to Frame UI</h1>
      <Input placeholder="Enter your name" />
      <Button>Submit</Button>
    </Card>
  );
}
```

### Theming

Frame UI supports both light and dark themes out of the box:

```tsx
import { darkTheme } from '@frame-ui/theme';

// Add dark mode support
document.documentElement.classList.toggle(darkTheme);
```

### Spacing Scale

Frame UI uses a consistent spacing scale for layouts and components:

| Token | Value | Usage |
|-------|-------|-------|
| **xs** | 4px | Minimal spacing, gaps |
| **sm** | 8px | Small components, margins |
| **md** | 12px | Standard padding, gaps |
| **lg** | 16px | Component padding, margins |
| **xl** | 24px | Section margins, gaps |

### Typography

Frame UI uses **Inter** as the default font family with responsive font sizes:

| Scale | Size | Usage |
|-------|------|-------|
| **xs** | 12px | Small labels, hints |
| **sm** | 14px | Body text, secondary content |
| **md** | 16px | Primary body text |
| **lg** | 18px | Headings, larger text |
| **xl** | 20px | Large headings |

## 🎯 Key Features

- **60+ Production-Ready Components**: A comprehensive library of accessible, reusable UI components
- **TypeScript First**: Full type safety with excellent IDE support
- **Zero-Runtime Styling**: Vanilla Extract CSS for optimal performance
- **Dark Mode Support**: Built-in light and dark theme support
- **Fully Accessible**: WCAG 2.1 AA compliant with ARIA labels and keyboard navigation
- **Responsive Design**: Mobile-first components that work on all screen sizes
- **Design Tokens**: Consistent spacing, colors, typography, and shadows
- **No Dependencies**: Minimal external dependencies for maximum flexibility

## 💡 Common Tasks

### Creating a Form

```tsx
import { Input, Button, Label, Field } from '@frame-ui/ui';

export function LoginForm() {
  return (
    <form>
      <Field>
        <Label htmlFor="email">Email</Label>
        <Input id="email" type="email" placeholder="you@example.com" />
      </Field>
      <Field>
        <Label htmlFor="password">Password</Label>
        <Input id="password" type="password" placeholder="••••••••" />
      </Field>
      <Button type="submit">Sign In</Button>
    </form>
  );
}
```

### Building a Card Layout

```tsx
import { Card, Badge, Button } from '@frame-ui/ui';

export function ProductCard() {
  return (
    <Card style={{ maxWidth: '300px' }}>
      <Badge>New</Badge>
      <h2>Amazing Product</h2>
      <p>Discover why developers love this product.</p>
      <Button>Learn More</Button>
    </Card>
  );
}
```

### Using the Theme

```tsx
import { vars } from '@frame-ui/theme';

export function StyledComponent() {
  return (
    <div style={{
      padding: vars.space.lg,
      background: vars.color.surface,
      borderRadius: vars.radius.md,
      color: vars.color.text,
    }}>
      Content with design system tokens
    </div>
  );
}
```
