# `@frame-ui/storybook`

Storybook app for local component development.

## Run from this folder

```bash
cd apps/storybook
pnpm dev
```

## Build from this folder

```bash
cd apps/storybook
pnpm build
```

## Same commands from repo root

```bash
pnpm --filter @frame-ui/storybook dev
pnpm --filter @frame-ui/storybook build
```
