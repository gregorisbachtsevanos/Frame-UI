# `@frame-ui/ui`

React UI component package.

## Run from this folder

```bash
cd packages/ui
pnpm dev
```

`pnpm dev` starts Storybook from this package.

## Build from this folder

```bash
cd packages/ui
pnpm build
```

## Lint and test from this folder

```bash
cd packages/ui
pnpm lint
pnpm test
```

## Same commands from repo root

```bash
pnpm --filter @frame-ui/ui dev
pnpm --filter @frame-ui/ui build
pnpm --filter @frame-ui/ui lint
pnpm --filter @frame-ui/ui test
```
